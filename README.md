# MFCPhoneBook

1. 기본 정보 확인
  * 정보 View 더블 클릭
2. 추가
  * 메뉴 View 이용
3. 삭제
  * 메뉴 View 이용  
4. 편집

5. 검색
  * 입력 후 버튼 클릭

  * csv 파일

  * csv 파일
